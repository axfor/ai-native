# 统一认证中心 — 需求文档

> **项目名称**：AuthHub — 业界主流认证方式 Demo
> **项目目标**：构建一个完整的认证中心，集成业界主流的认证/授权方案，作为团队实际项目的认证模块参考实现。

---

## 一、业务背景

企业级应用需要支持多种认证方式以满足不同场景需求：
- 内部系统用 **用户名密码 + JWT**
- 面向 C 端产品用 **OAuth2 社交登录**（GitHub、Google、微信）
- 企业客户用 **SSO 单点登录**（SAML / OIDC）
- 高安全场景用 **多因素认证（MFA/2FA）**
- 开放 API 用 **API Key + HMAC 签名**

## 二、功能需求

### 2.1 基础认证模块

| ID | 功能 | 优先级 | 说明 |
|----|------|--------|------|
| F-001 | 用户名密码注册 | P0 | 密码强度校验、bcrypt 加密存储 |
| F-002 | 用户名密码登录 | P0 | 返回 JWT Access Token + Refresh Token |
| F-003 | Token 刷新 | P0 | 用 Refresh Token 换取新的 Access Token |
| F-004 | 登出 | P0 | Token 黑名单机制（Redis） |
| F-005 | 密码重置 | P1 | 邮箱验证码方式 |

### 2.2 OAuth2 社交登录

| ID | 功能 | 优先级 | 说明 |
|----|------|--------|------|
| F-010 | GitHub OAuth2 登录 | P0 | Authorization Code Flow |
| F-011 | Google OAuth2 登录 | P1 | Authorization Code Flow + PKCE |
| F-012 | 微信 OAuth2 登录 | P1 | 适配微信特有的 OAuth 流程 |
| F-013 | 账号绑定 | P1 | 社交账号与本地账号关联/解绑 |

### 2.3 JWT Token 管理

| ID | 功能 | 优先级 | 说明 |
|----|------|--------|------|
| F-020 | Access Token 签发 | P0 | RS256 非对称加密，15 分钟过期 |
| F-021 | Refresh Token 签发 | P0 | 7 天过期，单设备单 Token |
| F-022 | Token 校验中间件 | P0 | 提取、验签、过期检查、注入用户信息 |
| F-023 | Token 黑名单 | P0 | Redis 存储已撤销的 Token |

### 2.4 RBAC 权限控制

| ID | 功能 | 优先级 | 说明 |
|----|------|--------|------|
| F-030 | 角色管理 | P0 | admin / user / guest 三级角色 |
| F-031 | 权限守卫中间件 | P0 | 基于角色的路由级权限控制 |
| F-032 | 资源级权限 | P1 | 用户只能操作自己的资源 |

### 2.5 多因素认证（MFA）

| ID | 功能 | 优先级 | 说明 |
|----|------|--------|------|
| F-040 | TOTP 双因素认证 | P1 | 兼容 Google Authenticator / Authy |
| F-041 | MFA 启用/禁用 | P1 | 用户自主开启关闭 |
| F-042 | 恢复码 | P1 | 生成一次性恢复码，用于 MFA 设备丢失 |

### 2.6 API Key 认证

| ID | 功能 | 优先级 | 说明 |
|----|------|--------|------|
| F-050 | API Key 签发 | P1 | 为第三方应用签发 API Key |
| F-051 | HMAC 签名验证 | P1 | 防篡改、防重放 |
| F-052 | 限流 | P1 | 按 API Key 限制请求频率 |

## 三、非功能需求

| 维度 | 要求 |
|------|------|
| **安全** | OWASP Top 10 防护、密码 bcrypt 12 轮、HTTPS Only |
| **性能** | 登录接口 < 200ms（P99）、Token 校验 < 5ms |
| **可用性** | Redis 不可用时降级为内存 Token 黑名单 |
| **可观测** | 登录/登出/失败事件记录审计日志 |
| **合规** | 密码不明文存储、Token 不含敏感信息、支持账号删除 |
